# Mobile Platform

React Native application for iOS and Android devices.

## Contents
- Expo-managed React Native app
- Mobile navigation (React Navigation)
- Platform-specific components
- Native module integrations

## Key Relationships
- Uses `@umemee/ui-mobile` for native UI components
- Shares business logic from `@umemee/utils`
- Connects via `@umemee/api-client` for data
