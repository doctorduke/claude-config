# Shared

Shared packages and utilities used across all platforms and applications.

## Contents

- **types/** - TypeScript type definitions and interfaces
- **config/** - Configuration management and environment settings
- **utils/** - Core utility functions and common business logic
- **api-client/** - API client for backend communication
- **ui-mobile/** - Shared UI components for mobile platforms
- **ui-web/** - Shared UI components for web platforms

These packages provide common functionality that can be imported by any platform or service in the monorepo.
